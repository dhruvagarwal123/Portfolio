## 🛠️ How to Use

1. **Clone the repository:**
   ```sh
   git clone https://github.com/dhruvagarwal123/portfolio.git
   ```
2. **Open the project folder:**
   ```sh
   cd portfolio
   ```
3. **Open `index.html` in your web browser.**
